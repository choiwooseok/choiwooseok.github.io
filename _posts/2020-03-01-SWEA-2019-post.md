* 개인적인 풀이. 문제시 삭제 예정, 더 좋은 방법 공유 환영

```cpp
#include <iostream>
#include <cmath>

int main() {
	int n;
    std::cin >> n;
    for(int i = 0; i <= n; i++) {
    	std::cout << std::pow(2, i) << " ";
    }
    return 0;
}
```
