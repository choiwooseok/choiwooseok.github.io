* 개인적인 풀이. 문제시 삭제 예정, 더 좋은 방법 공유 환영

```cpp
#include <algorithm>
#include <iostream>
#include <string>

int main() {
	std::string in;
    std::cin >> in;
    std::transform(in.begin(), in.end(), in.begin(), ::toupper);
    std::cout << in;
    return 0;
}
```
